Quantum-safe Encrypted Search over Cloud Data

Funding Agency: ANRF (Anusandhan National Research Foundation) Funding Scheme: PM ECRG (Early Career Research Grant) Status: Approved in April 2026 Total Budget: ₹58.7 lakh

This project focuses on designing quantum-safe encrypted search mechanisms for secure cloud data storage and retrieval. It aims to develop cryptographic techniques that preserve data confidentiality while enabling efficient search over encrypted datasets, resilient against both classical and quantum adversaries.

Key Outcomes
  • Design of post-quantum secure searchable encryption schemes.
  • Development of efficient protocols for keyword and structured queries over encrypted cloud data.
  • Performance evaluation of proposed schemes in realistic cloud environments.
  • Prototype implementation for secure and scalable encrypted search systems.
